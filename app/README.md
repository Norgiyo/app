# 1Cup App (Flutter)

Cliente movil de 1Cup.  
Stack principal: Flutter + Firebase Auth + Firestore.

## Requisitos

- Flutter 3.x
- Dart 3.x
- Firebase CLI (para emuladores cuando aplique)

## Setup rapido

```powershell
cd app
flutter pub get
flutter run
```

## Calidad minima

```powershell
cd app
flutter analyze
flutter test
```
